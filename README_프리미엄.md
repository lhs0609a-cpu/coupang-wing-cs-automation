## ✨ 주요 특징

### 🎨 최신 트렌드 UI
- **유리 효과 (Glassmorphism)** 디자인
- **그라데이션 컬러** 스킴
- **실시간 애니메이션** 효과
- **반응형 레이아웃**
- **다크 모드** 최적화

### 🤖 완전 자동화
- ✅ **원클릭 실행**: 한 번의 클릭으로 모든 것 자동 처리
- ✅ **자동 서버 시작**: 포트 자동 탐지 및 서버 시작
- ✅ **스마트 로그인**: 쿠팡윙 자동 로그인
- ✅ **AI 답변 생성**: ChatGPT로 자동 답변 작성
- ✅ **실시간 모니터링**: 통계 및 로그 실시간 업데이트

## 🎯 주요 기능

### 1. 완전 자동화 실행
```
한 번의 클릭으로:
✅ 환경 변수 자동 저장
✅ 빈 포트 자동 탐지 (8000-8100)
✅ 백엔드 서버 자동 시작
✅ 서버 연결 자동 확인
✅ 쿠팡윙 자동 로그인
✅ 모든 시간대 탭 자동 순회
✅ 문의 자동 수집 및 답변
✅ 통계 실시간 업데이트
```

### 2. 시간대별 탭 처리
- 72시간~30일 이내
- 24~72시간
- 24시간 이내

### 3. AI 답변 생성
- ChatGPT-4 사용
- 상품명과 문의 내용 분석
- 자연스러운 고객 응대 답변

### 4. 실시간 모니터링
- 총 문의 수
- 답변 완료 수
- 실패 수
- 건너뜀 수

## 🔧 설치 및 설정

### 1. 의존성 설치
```bash
# 이미 설치되어 있습니다
python -m pip install customtkinter pillow requests
```

### 2. 환경 설정
GUI에서 입력하면 자동으로 `.env` 파일에 저장됩니다:
```env
COUPANG_WING_USERNAME=your_email@example.com
COUPANG_WING_PASSWORD=your_password
OPENAI_API_KEY=sk-your-api-key-here
OPENAI_MODEL=gpt-4
```

## 🎮 사용 방법

### 기본 사용법

#### 1. 처음 사용 (브라우저 보면서)
```
1. 쿠팡윙_자동화_실행.bat 실행
2. 로그인 정보 입력
3. "Headless 모드" 체크 해제
4. "▶️ 자동화 시작" 클릭
5. 브라우저가 열리면서 자동 진행
6. 통계와 로그로 확인
```

#### 2. 정기 사용 (빠른 실행)
```
1. 쿠팡윙_자동화_실행.bat 실행
2. 정보가 자동으로 채워짐
3. "Headless 모드" 체크 유지
4. "▶️ 자동화 시작" 클릭
5. 백그라운드에서 자동 처리
6. 통계로 결과 확인
```

#### 3. 로그인만 테스트
```
1. GUI 실행
2. 정보 입력
3. "🔍 로그인 테스트" 클릭
4. 로그에서 결과 확인
```

## 🐛 문제 해결

### GUI 실행 오류
```bash
# CustomTkinter 재설치
python -m pip install --upgrade customtkinter

# 직접 실행
python coupang_wing_gui.py
```

### 서버 시작 실패
```
원인: 포트가 모두 사용 중
해결: 다른 프로그램 종료 후 재시도

원인: 가상환경 없음
해결: python -m venv venv
```

### 로그인 실패
```
확인 사항:
1. 아이디/비밀번호 정확성
2. Headless OFF로 브라우저 확인
3. 쿠팡윙 웹사이트 접속 가능 여부
4. 방화벽 설정
```

## 🚀 기술 스택

### Frontend (GUI)
- **CustomTkinter**: 모던한 GUI 프레임워크
- **Tkinter**: 기본 GUI 라이브러리

### Backend
- **FastAPI**: 비동기 웹 프레임워크
- **Uvicorn**: ASGI 서버
- **Selenium**: 웹 자동화
- **ChromeDriver**: 브라우저 드라이버

### AI
- **OpenAI GPT-4**: 답변 생성
- **langchain**: (선택사항) AI 체인
