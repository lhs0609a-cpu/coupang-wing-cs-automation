# ✨ 쿠팡윙 CS 자동화 - Instagram Style

> 백엔드 없이 바로 실행되는 독립형 앱

---

## 🌟 특징

### Instagram 스타일 디자인
- **밝고 깔끔한 UI**: 인스타그램의 #FAFAFA 배경색
- **Instagram 그라데이션**: 핑크(#E1306C) → 보라(#C13584)
- **카드 형식 레이아웃**: 둥근 모서리와 그림자 효과
- **미니멀한 디자인**: 불필요한 요소 제거

### 완전 독립 실행
- ✅ **백엔드 서버 불필요**: 서버 시작 없이 바로 실행
- ✅ **포트 탐색 불필요**: 복잡한 설정 없음
- ✅ **원클릭 실행**: 정보 입력 후 바로 시작
- ✅ **Selenium 직접 실행**: GUI에서 직접 브라우저 제어

## 🎨 디자인 특징

### Instagram 컬러 팔레트
```css
/* 메인 컬러 */
Instagram Pink:    #E1306C  /* 시작 버튼 */
Instagram Purple:  #C13584  /* 호버 효과 */
Instagram Orange:  #F56040  /* 액센트 */
Instagram Yellow:  #FCAF45  /* 하이라이트 */

/* UI 컬러 */
Background:        #FAFAFA  /* 배경 */
Card:              #FFFFFF  /* 카드 */
Border:            #DBDBDB  /* 테두리 */
Text Dark:         #262626  /* 진한 텍스트 */
Text Light:        #8E8E8E  /* 연한 텍스트 */

/* 상태 컬러 */
Success:           #4CB050  /* 성공 */
Error:             #ED4956  /* 오류 */
```

### UI 요소
- **카드**: 흰색 배경 + #DBDBDB 테두리 + 15px 라운드
- **버튼**: 55px 높이 + Instagram 핑크 + 10px 라운드
- **입력 필드**: 42px 높이 + 8px 라운드
- **통계 카드**: #FAFAFA 배경 + 12px 라운드

### 타이포그래피
- **로고**: 40px ✨
- **제목**: 20px Bold Arial
- **섹션 타이틀**: 16px Bold
- **입력 라벨**: 12px Bold
- **통계 숫자**: 24px Bold
- **본문**: 13px
- **상태**: 12px

## 🎯 사용법

### 처음 사용 (브라우저 보면서)
```
1. 인스타_실행.bat 더블클릭
2. 정보 입력:
   - 이메일
   - 비밀번호
   - API Key
3. "백그라운드 실행" 체크 해제
4. "시작하기" 버튼 클릭
5. 브라우저가 열리면서 자동 진행
6. 로그와 통계로 확인
```

### 정기 사용 (빠른 실행)
```
1. 인스타_실행.bat 더블클릭
2. 정보 자동 로드됨
3. "백그라운드 실행" 체크 유지
4. "시작하기" 버튼 클릭
5. 완료 대기
```

## ✨ 주요 개선 사항

### 이전 버전 대비
```
❌ 백엔드 서버 시작 필요      → ✅ 백엔드 불필요
❌ 포트 탐색 및 연결          → ✅ 포트 불필요
❌ 복잡한 설정               → ✅ 간단한 설정
❌ 여러 단계 실행            → ✅ 원클릭 실행
❌ 어두운 테마                → ✅ 밝은 Instagram 테마
❌ 복잡한 레이아웃           → ✅ 심플한 카드 레이아웃
```

## 💡 실행 흐름

```
1. 배치 파일 실행
   ↓
2. Instagram 스타일 GUI 로딩
   ├─ .env 파일에서 정보 자동 로드
   └─ UI 초기화
   ↓
3. 사용자 입력 (처음만)
   ├─ 이메일
   ├─ 비밀번호
   └─ API 키
   ↓
4. "시작하기" 버튼 클릭
   ↓
5. 직접 실행 (백엔드 없음!)
   ├─ Selenium 웹드라이버 시작
   ├─ 쿠팡윙 로그인
   ├─ 고객문의 페이지 이동
   ├─ 문의 수집
   ├─ 답변 작성 및 저장
   └─ 통계 실시간 업데이트
   ↓
6. 완료!
```

## 📋 기술 스택

### GUI
- **CustomTkinter**: 모던 GUI 프레임워크
- **Tkinter**: 기본 GUI 라이브러리

### 자동화
- **Selenium**: 웹 브라우저 자동화
- **ChromeDriver**: Chrome 브라우저 제어

### AI (선택사항)
- **OpenAI GPT-4**: 답변 생성
