# 🚀 쿠팡윙 CS 자동화 - GUI 사용 가이드

## 트렌디한 GUI로 쉽게 사용하기!

### GUI 실행 방법

```bash
# 방법 1: 배치 파일 더블클릭
GUI_실행.bat

# 방법 2: Python 직접 실행
python gui_launcher.py
```

---

## 📱 GUI 화면 구성

### 왼쪽 패널 - 설정 및 컨트롤

#### 1. 🔐 로그인 정보
- **아이디**: 쿠팡윙 로그인 이메일
- **비밀번호**: 쿠팡윙 로그인 비밀번호
- **OpenAI API Key**: ChatGPT API 키 (sk-로 시작)

💡 **자동 저장**: 입력한 정보는 자동으로 `.env` 파일에 저장되어 다음 실행 시 자동으로 불러옵니다!

#### 2. ⚙️ 실행 설정
- **Headless 모드**: 체크하면 브라우저 창이 보이지 않음 (빠르고 효율적)
  - ✅ 체크: 백그라운드 실행
  - ❌ 체크 해제: 브라우저 창 보면서 실행 (디버깅용)

- **자동 모드**: 5분마다 자동으로 문의 확인 및 답변
  - ✅ 체크: 5분마다 자동 실행
  - ❌ 체크 해제: 수동으로 버튼 클릭 시에만 실행

#### 3. 🎮 컨트롤 버튼

**🔍 로그인 테스트** (회색 버튼)
- 로그인 정보가 올바른지 테스트
- 브라우저가 열리고 실제로 로그인 시도
- 성공/실패 여부를 로그에 표시

**🌐 서버 시작** (청록색 버튼)
- 백엔드 서버 시작
- 자동화를 실행하려면 서버가 먼저 실행되어야 함
- 상태: "🟢 서버: 실행 중" 으로 변경됨

**▶️ 자동화 시작** (주황색 대형 버튼)
- V2 자동화 시스템 실행
- 모든 시간대 탭을 확인하고 문의 답변
- 실행 중에는 버튼이 비활성화됨

**⏸️ 중지** (회색 버튼)
- 자동화 중지 요청
- 현재 처리 중인 작업은 완료 후 중지

### 오른쪽 패널 - 통계 및 로그

#### 📊 통계 카드
4개의 실시간 통계 카드 표시:
- **총 문의**: 발견된 전체 문의 수
- **답변 완료**: 성공적으로 답변한 문의 수
- **실패**: 답변 실패한 문의 수
- **건너뜀**: 건너뛴 문의 수

#### 📝 실행 로그
- 실시간으로 모든 작업 진행 상황 표시
- 타임스탬프와 함께 기록
- 자동 스크롤로 최신 로그 확인 가능
- 이모지로 직관적인 상태 표시:
  - ✅ 성공
  - ❌ 오류
  - ℹ️ 정보
  - ⏳ 진행 중
  - 🔍 확인 중

## 🎯 사용 시나리오

### 시나리오 1: 처음 사용하기

1. **GUI_실행.bat** 더블클릭
2. 로그인 정보 입력:
   - 아이디: `your_email@example.com`
   - 비밀번호: `your_password`
   - API Key: `sk-...`
3. **🔍 로그인 테스트** 버튼 클릭
   - Headless 체크 해제 (브라우저 보면서 확인)
4. 로그에서 "✅ 로그인 성공!" 확인
5. **🌐 서버 시작** 버튼 클릭
6. 서버 상태가 "🟢 서버: 실행 중" 으로 변경될 때까지 대기 (약 10초)
7. **▶️ 자동화 시작** 버튼 클릭
8. 통계와 로그를 확인하며 진행 상황 모니터링

### 시나리오 2: 빠른 실행 (설정 저장 후)

1. **GUI_실행.bat** 더블클릭
2. 로그인 정보가 자동으로 채워져 있음 (이전에 저장됨)
3. **Headless 모드** 체크 (빠른 실행)
4. **🌐 서버 시작** 버튼 클릭
5. 서버 상태 확인 후 **▶️ 자동화 시작** 클릭
6. 완료 대기

### 시나리오 3: 자동 모드 (완전 자동화)

1. **GUI_실행.bat** 더블클릭
2. **자동 모드 (5분마다 체크)** 체크박스 활성화
3. **Headless 모드** 체크
4. **🌐 서버 시작** 버튼 클릭
5. **▶️ 자동화 시작** 버튼 클릭
6. GUI 창을 열어두면 5분마다 자동으로 문의 확인 및 답변!
   - 새 문의가 있으면 자동 답변
   - 없으면 5분 후 다시 체크

## 💡 사용 팁

### 1. 처음 사용할 때
- **Headless 모드 체크 해제**하여 브라우저를 보면서 실행
- 로그인 과정과 자동화 과정을 직접 확인
- 문제가 없는지 확인 후 다음부터 Headless 모드 사용

### 2. 정기적으로 사용할 때
- **Headless 모드 체크**하여 빠르게 실행
- 로그 창을 통해 진행 상황 모니터링
- 통계 카드로 처리 현황 확인

### 3. 완전 자동화
- **자동 모드**를 활성화하고 GUI를 열어둠
- 5분마다 자동으로 체크하여 새 문의에 답변
- 컴퓨터가 켜져 있는 동안 계속 작동

### 4. 여러 번 실행
- 한 번 정보를 입력하면 `.env` 파일에 저장됨
- 다음 실행 시 자동으로 불러와짐
- 필요시 GUI에서 직접 수정 가능

## 🔧 문제 해결

### GUI가 열리지 않음
```bash
# CustomTkinter 설치 확인
python -m pip install customtkinter pillow

# GUI 실행
python gui_launcher.py
```

### 서버가 시작되지 않음
```bash
# 수동으로 서버 시작
START.bat

# 또는
python run_server.py
```

### 로그인 테스트 실패
1. Headless 모드 체크 해제
2. 브라우저가 열리는지 확인
3. 아이디/비밀번호가 정확한지 확인
4. 로그 창에서 오류 메시지 확인

### 자동화 실행 시 오류
1. 서버가 실행 중인지 확인 ("🟢 서버: 실행 중")
2. 로그인 정보가 올바른지 확인
3. OpenAI API 키가 유효한지 확인
4. 로그 창에서 상세 오류 확인

## 🌟 주요 기능

### ✅ 자동 저장
- 입력한 로그인 정보는 자동으로 `.env` 파일에 저장
- 다음 실행 시 자동으로 불러옴

### ✅ 실시간 모니터링
- 로그 창으로 실시간 진행 상황 확인
- 통계 카드로 처리 현황 한눈에 파악

### ✅ 서버 상태 체크
- 5초마다 자동으로 서버 연결 상태 확인
- 상태 표시: 🟢 실행 중 / ⚪ 연결 안됨

### ✅ 안전한 실행
- 입력값 검증으로 오류 방지
- 스레드 기반 비동기 실행으로 GUI 멈춤 현상 없음

### ✅ 사용자 친화적
- 큰 버튼과 명확한 라벨
- 이모지로 직관적인 상태 표시
- 타임스탬프가 포함된 로그
