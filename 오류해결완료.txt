🚀 실행 방법
-----------
1. Windows 탐색기에서 이 폴더를 엽니다:
   C:\Users\u\coupang-wing-cs-automation\

2. "최종실행파일1.bat" 파일을 더블클릭합니다

3. 자동으로 실행되는 것을 확인합니다:

💡 팁
-----
• 처음 실행 시 Windows 방화벽 경고가 뜰 수 있습니다
  → "액세스 허용" 클릭

• 포트가 이미 사용 중이면 자동으로 다른 포트를 찾습니다

• 연결이 실패하면 자동으로 재시도합니다 (최대 10회)

• 브라우저가 자동으로 열리지 않으면 수동으로 주소 입력:
  http://localhost:3000 (또는 표시된 포트)


⚠️ 주의사항
-----------
• Python이 설치되어 있어야 합니다
• backend와 frontend 폴더가 같은 위치에 있어야 합니다
• venv 가상환경이 설정되어 있어야 합니다
• Node.js와 npm이 설치되어 있어야 합니다


🔧 문제 발생 시
-------------
Q: 배치 파일을 실행해도 아무 일도 일어나지 않습니다
A: Python이 설치되어 있고 PATH에 등록되어 있는지 확인하세요

Q: "python을 찾을 수 없습니다" 오류가 발생합니다
A: Python 설치 후 시스템 환경 변수에 Python 경로를 추가하세요

Q: 서버가 시작되지 않습니다
A: backend/frontend 폴더가 올바른 위치에 있는지 확인하세요
