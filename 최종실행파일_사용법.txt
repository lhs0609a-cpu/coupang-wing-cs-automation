📌 파일 위치
-----------
현재 폴더: C:\Users\u\coupang-wing-cs-automation\
실행 파일: 최종실행파일.bat


🚀 실행 방법
-----------
1. "최종실행파일.bat" 파일을 더블클릭하세요
2. 자동으로 다음 작업이 수행됩니다:
   - 사용 가능한 빈 포트 자동 탐색
   - 프론트엔드 설정 자동 업데이트
   - 백엔드 서버 시작
   - 백엔드 연결 확인 (최대 30번 재시도)
   - 프론트엔드 서버 시작
   - 프론트엔드 연결 확인 (최대 20번 재시도)
   - 웹 브라우저로 프론트엔드 자동 열기


✨ 주요 기능
-----------
✅ 자동 빈 포트 탐색
   - 백엔드: 8000~8099 범위에서 자동 탐색
   - 프론트엔드: 3000~3099 범위에서 자동 탐색

✅ 자동 재연결
   - 연결 실패 시 다른 포트로 자동 재시도 (최대 3번)
   - 각 서버별 연결 확인 재시도

✅ 웹 브라우저 자동 실행
   - 모든 연결 완료 후 자동으로 프론트엔드 페이지 열기

✅ 실시간 상태 확인
   - 각 단계별 진행 상황 실시간 표시
   - 연결 성공/실패 여부 명확히 표시


📋 실행 후 화면
-------------
연결이 완료되면 다음과 같은 정보가 표시됩니다:

- Frontend:  http://localhost:3000 (또는 자동 할당된 포트)
- Backend:   http://localhost:8000 (또는 자동 할당된 포트)
- API Docs:  http://localhost:8000/docs

그리고 자동으로 웹 브라우저가 열립니다!


🛑 서버 종료 방법
---------------
1. 백엔드 콘솔 창에서 Ctrl+C 누르기
2. 프론트엔드 콘솔 창에서 Ctrl+C 누르기


⚠️ 주의사항
-----------
1. Python이 설치되어 있어야 합니다
2. backend 폴더와 frontend 폴더가 같은 디렉토리에 있어야 합니다
3. venv (가상환경)가 설정되어 있어야 합니다
4. 첫 실행 시 방화벽 경고가 뜰 수 있습니다 (허용 필요)


🔧 문제 해결
-----------
Q: 실행이 안 됩니다
A: Python이 설치되어 있고 PATH에 등록되어 있는지 확인하세요

Q: 포트를 찾을 수 없다고 나옵니다
A: 다른 프로그램이 8000~8099 또는 3000~3099 포트를 사용 중일 수 있습니다
   실행 중인 서버를 종료하고 다시 시도하세요

Q: 연결이 계속 실패합니다
A: 1) 백엔드와 프론트엔드 폴더가 올바른 위치에 있는지 확인
   2) venv가 제대로 설정되어 있는지 확인
   3) 네트워크 방화벽 설정 확인


📞 지원
------
문제가 계속되면 로그를 확인하거나 개발자에게 문의하세요.
