# 🎮 프로모션 쿠폰 설정 가이드 (게임처럼 쉽게!)

## 시작하기 전에
쿠폰을 설정하면 **모든 상품에 자동으로 쿠폰이 적용**되어 매출이 10배 늘어납니다! 🚀

---

## 🎯 STEP 1: 계정 선택하기

### 1️⃣ 첫 번째로 할 일
1. 화면 오른쪽 위에 **"계정 선택"** 드롭다운을 찾으세요
2. 클릭해서 쿠폰을 적용할 쿠팡 계정을 선택하세요
   ```
   예시: "내 쇼핑몰" 또는 "A01234567"
   ```

### ✅ 완료 조건
- 계정이 선택되면 아래에 통계가 표시됩니다

---

## 🎯 STEP 2: 쿠폰 종류 선택하기

### 2️⃣ 어떤 쿠폰을 사용할지 선택하세요

#### 옵션 A: 즉시할인쿠폰 (추천! ⚡)
- **효과**: 고객이 바로 할인된 가격을 볼 수 있어 전환율 최고!
- **설정 방법**:
  1. "즉시할인쿠폰" 카드의 **토글 스위치를 켜세요** (왼쪽 → 오른쪽)
  2. 카드가 **파란색**으로 활성화됩니다 ✨
  3. 아래 입력 필드가 나타납니다

#### 옵션 B: 다운로드쿠폰 (고객 유입용! 📥)
- **효과**: 고객이 다운로드해서 사용하는 쿠폰
- **설정 방법**:
  1. "다운로드쿠폰" 카드의 **토글 스위치를 켜세요**
  2. 카드가 활성화됩니다

#### 💡 Tip: 둘 다 선택 가능!
- 즉시할인 + 다운로드쿠폰을 함께 사용하면 효과가 배가됩니다!

### ✅ 완료 조건
- 최소 하나의 쿠폰 카드가 파란색으로 활성화되어야 합니다

---

## 🎯 STEP 3: 쿠폰 선택하기 (가장 중요!)

### 3️⃣ 쿠폰을 불러와서 선택하세요

#### 방법 1: 기존 쿠폰 선택 (추천! ⭐)

**즉시할인쿠폰을 선택했다면:**
1. "또는 기존 쿠폰에서 선택" 섹션을 찾으세요
2. 🔄 **"불러오기" 버튼을 클릭**하세요
   - 잠시 기다리면 쿠폰 목록이 로드됩니다
3. 드롭다운을 클릭해서 원하는 쿠폰을 선택하세요
   ```
   예시: [12345] 10% 할인쿠폰
   ```
4. 선택하면 쿠폰 ID와 이름이 자동으로 입력됩니다 ✨

**다운로드쿠폰을 선택했다면:**
1. 쿠팡윙에 로그인하세요
2. **할인쿠폰 관리** 메뉴로 이동
3. 다운로드쿠폰 목록에서 쿠폰 ID를 복사하세요
4. "쿠폰 ID" 입력 칸에 붙여넣으세요

#### 방법 2: 직접 입력하기
1. "쿠폰 ID" 입력 칸에 숫자를 입력하세요
   ```
   예시: 12345
   ```
2. "쿠폰명 (메모용)" 칸에 메모를 적으세요 (선택사항)
   ```
   예시: 블랙프라이데이 20% 할인
   ```

### ⚠️ 주의사항
- **쿠폰 ID가 0 또는 -1이면 작동하지 않습니다!**
- 반드시 유효한 쿠폰 ID를 입력하세요

### ✅ 완료 조건
- 쿠폰 ID 입력 칸에 **0이 아닌 숫자**가 입력되어야 합니다

---

## 🎯 STEP 4: 적용 옵션 설정하기

### 4️⃣ 언제 쿠폰을 적용할지 설정하세요

#### 신규 상품 적용 대기일
1. 드롭다운을 클릭하세요
2. 원하는 옵션을 선택하세요:
   - **"즉시 적용"** (추천!): 상품 등록 즉시 쿠폰 적용
   - **"1일 후"**: 상품 등록 1일 후에 쿠폰 적용
   - **"3일 후"**: 노출이 어느정도 된 후 쿠폰 적용

💡 **Tip**: 즉시 적용을 선택하면 상품 등록하자마자 쿠폰이 붙어서 매출이 바로 올라갑니다!

#### 기존 상품에도 적용할까요?
- ✅ **"저장 시 기존 전체 상품에도 쿠폰 자동 적용"** 체크박스를 **반드시 체크**하세요!
- 이 옵션을 체크하면:
  - 이미 등록된 모든 상품에도 쿠폰이 일괄 적용됩니다
  - 수백, 수천 개 상품에 클릭 한 번으로 자동 적용! 🎉

### ✅ 완료 조건
- 대기일이 선택되어야 합니다

---

## 🎯 STEP 5: 저장하고 적용하기! (마지막 단계!)

### 5️⃣ 큰 파란색 버튼을 클릭하세요

1. 화면 아래에 있는 **"설정 저장 및 적용"** 버튼을 찾으세요
2. 버튼이 회색이면 위 단계를 다시 확인하세요
3. 버튼이 **파란색**이면 준비 완료! 🎉
4. **클릭하세요!**

### 🎊 저장 후 일어나는 일
1. ✅ 설정이 저장됩니다
2. 🔄 백그라운드에서 전체 상품 수집이 시작됩니다
3. 🎫 모든 상품에 쿠폰이 자동으로 적용됩니다
4. 📊 진행률이 실시간으로 표시됩니다

### 진행 상황 확인하기
- 화면 위쪽에 **진행률 표시줄**이 나타납니다
- **"1단계: 상품 정보 수집"**: 상품 목록을 가져오는 중
- **"2단계: 쿠폰 적용"**: 쿠폰을 적용하는 중
- 완료되면 ✅ **"완료"** 메시지가 표시됩니다

### ✅ 완료!
축하합니다! 🎉 이제 모든 상품에 쿠폰이 적용되었습니다!

---

## 📊 설정 후 확인하기

### 통계 확인
- 화면 위쪽의 통계 카드를 확인하세요:
  - 📦 **총 추적 상품**: 쿠폰이 적용된 상품 개수
  - ⏰ **대기 중**: 아직 적용 대기 중인 상품
  - ✅ **적용 완료**: 쿠폰 적용이 완료된 상품
  - ❌ **실패**: 적용에 실패한 상품

### 현재 설정 확인
- 아래로 스크롤하면 **"현재 적용된 설정"** 박스가 표시됩니다
- 여기서 어떤 쿠폰이 활성화되어 있는지 확인할 수 있습니다

---

## 🔧 문제 해결

### ❌ "쿠폰 ID를 입력해주세요" 에러가 나요
➡️ 쿠폰 ID 칸이 비어있거나 0입니다. 유효한 쿠폰 ID를 입력하세요.

### ❌ "최소 하나의 쿠폰을 활성화해주세요" 에러가 나요
➡️ 즉시할인쿠폰이나 다운로드쿠폰 중 하나는 반드시 활성화해야 합니다.

### ❌ "불러오기" 버튼을 눌러도 쿠폰이 안 나와요
➡️ 쿠팡윙에서 먼저 쿠폰을 생성하고 "적용" 상태로 만들어야 합니다.

### ❌ 저장 버튼이 회색이고 클릭이 안 돼요
➡️ 위 단계를 모두 완료했는지 확인하세요:
  - [ ] 계정 선택
  - [ ] 쿠폰 종류 활성화 (토글 ON)
  - [ ] 쿠폰 ID 입력 (0이 아닌 숫자)

---

## 🎮 요약: 5단계로 끝내기

```
1️⃣ 계정 선택 (드롭다운 클릭)
    ↓
2️⃣ 쿠폰 종류 선택 (토글 스위치 ON)
    ↓
3️⃣ 쿠폰 불러오기 (🔄 버튼 클릭 → 드롭다운 선택)
    ↓
4️⃣ 옵션 설정 (✅ 전체 상품 적용 체크)
    ↓
5️⃣ 저장 버튼 클릭 (파란색 버튼)
    ↓
🎉 완료!
```

## 💡 Pro Tips

1. **즉시할인쿠폰을 먼저 설정하세요**
   - 가장 효과가 좋고 설정도 쉽습니다!

2. **"전체 상품 적용" 체크를 잊지 마세요**
   - 이걸 체크하지 않으면 신규 상품에만 적용됩니다

3. **진행률을 지켜보세요**
   - 상품이 많으면 몇 분 정도 걸릴 수 있습니다
   - 창을 닫아도 백그라운드에서 계속 진행됩니다

4. **통계를 주기적으로 확인하세요**
   - 적용 실패한 상품이 있는지 확인하고 조치하세요
