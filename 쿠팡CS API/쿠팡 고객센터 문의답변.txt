쿠팡 고객센터 문의답변
API 적용 가능한 구매자 사용자 지역: 한국, 대만

판매자가 쿠팡 고객센터를 통해 접수된 문의에 대해 답변합니다.
문의의 상태가 '미답변(inquiryStatus:progress, partnerTransferStatus:requestAnswer)'인 경우에만 답변 작성 가능합니다.
고객센터 문의조회 API를 통해 해당 inquiryId, answerId를 확인한 뒤 답변 작성 부탁드립니다.
동일한 inquiry에 대해 중복해서 답변을 보내면 에러가 발생합니다.

※ 문의의 상태가 24시간 이상 ‘미답변’ 상태로 남아있는 경우,
쿠팡 시스템에서 자동으로 “[쿠팡 시스템] CS문의가 24시간 내 답변되지 않아, 쿠팡 약관 및 정책에 따라 상담사가 처리할 예정임을 알려드립니다.” 안내가 문의에 달리게 됩니다.
이 경우 안내와 동시에 문의 상태는 “답변완료“로 변경이 되므로 API를 통해 답변을 할 수 없으며
필요 시 [WING > 온라인 문의 > 쿠팡CS문의] 페이지를 통해 답변 작성 가능합니다.
Path
POST
/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/callCenterInquiries/{inquiryId}/replies

Example Endpoint
https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00010028/callCenterInquiries/1007837444/replies
