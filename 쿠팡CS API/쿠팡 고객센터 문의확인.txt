쿠팡 고객센터 문의확인
API 적용 가능한 구매자 사용자 지역: 한국, 대만

쿠팡이 상담완료한 업체이관 건으로 판매자의 확인이 필요한 상태(미확인 상태:TRANSFER)일 때 이 API를 이용하여 확인 처리합니다.
상담사에 의해 종료되었거나 확인요청 후 24시간이 경과된 문의는 더 이상 확인할 수 없습니다.
Path
POST

/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/callCenterInquiries/{inquiryId}/confirms

Example Endpoint
https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00010028/callCenterInquiries/1007837444/confirms

Request Parameters
Path Segment Parameter
Name	Required	Type	Description
vendorId	O	 	
vendor ID
inquiryId	O	 	
Inquiry ID
판매자가 어느 질문을 확인해야할 지를 입력합니다.
먼저 쿠팡 고객센터 문의조회 API를 사용하여 미확인 상태(TRANSFER)인 inquiry 값을 확인합니다.
Body Parameter
Name	Required	Type	Description
confirmBy	O 	String	
실사용자ID(쿠팡 Wing ID)
업체(Vendor)에 소속된 사용자ID
Request Example
{
  "confirmBy": "wanger"
}
Response Message
Name	Type	Description
code	Number	
서버 응답 코드
200/400/500
message	String	
서버 응답 메시지
If success,it shows OK
Response Example
{
  "code": "200",
  "message": "OK"
}
Error Spec
HTTP 상태 코드(오류 유형)	오류 메시지	해결 방법
400(요청변수확인)	상담이 종료된 문의입니다 - 522 	이미 확인이 완료되었거나 종료 또는 24시간이 경과한 문의인지 확인합니다. 종료된 문의에 중복으로 확인할 경우 에러가 발생합니다.
400 (요청변수확인)	The inquiry can't be answer. It can do only inquiryStatus:progress, partnerTransferStatus:requestAnswer	이미 확인이 완료되었거나 종료 또는 24시간이 경과한 문의인지 확인합니다. 종료된 문의에 중복으로 확인할 경우 에러가 발생합니다.
URL API Name
CONFIRM_INQUIRY
