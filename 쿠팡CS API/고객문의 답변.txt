상품별 고객문의 답변
API 적용 가능한 구매자 사용자 지역: 한국, 대만

고객문의(inquiryId)에 대해 답변할 수 있습니다
하나의 고객문의(inquiryId)에 답변할 수 있습니다. 먼저 상품별 고객문의 조회 API를 사용하여 inquiryId를 확인하기 바랍니다.

답변내용(content)을 JSON 형식에 맞게 작성하시기 바랍니다.
Path
POST
/v2/providers/openapi/apis/api/v4/vendors/{vendorId}/onlineInquiries/{inquiryId}/replies

Example Endpoint
https://api-gateway.coupang.com/v2/providers/openapi/apis/api/v4/vendors/A00010028/onlineInquiries/846/replies
Request Parameters
Path Segment Parameter
Name	Required	Type	Description
inquiryId	O	String	
Inquiry ID
판매자가 어느 질문을 답변할 지를 입력합니다.
먼저 상품별 고객문의 조회 API를 사용하여 inquiryId를 확인하기 바랍니다.
vendorId	O	String 	
판매자 ID
Body Parameter
Name	Required	Type	Description
content	O	String	
답변 내용
답변 내용 중 줄넘김을 위해서는 '\n'를 사용하시기 바랍니다.
vendorId	O	String	
판매자 ID
replyBy	O	String	
응답자 셀러포탈(WING) 아이디
Request Example
{
  "content": "안녕하세요\n블랙색상이 주문량이 많아 입고 지연중입니다.\n19일 입고예정으로 확인됩니다.",
  "vendorId": "A00010028",
  "replyBy": "wanger"
}

Error Spec
HTTP 상태 코드(오류 유형)	오류 메시지	해결 방법
400 (요청변수확인)	The value of replyBy (Responder Seller Portal (WING) ID) is incorrect. Please enter the correct wing ID or vendorId.	replyBy 파라메터 값으로 올바른 셀러포탈(WING) 아이디를 입력했는지 확인합니다. 잘못된 WING ID를 입력 시 오류가 발생합니다.
400 (요청변수확인)	삭제된 상품문의에는 더 이상 답변할 수 없습니다.	삭제된 상품문의글에는 답변을 입력하지 않도록 수정합니다.
400 (요청변수확인)	Could not read JSON: Illegal unquoted character ((CTRL-CHAR, code 13)): has to be escaped using backslash to be included in string value...	답변내용(content)을 JSON 형식에 맞게 작성했는지 확인합니다. newline, backslash, carriage return 등의 문자는 JSON 형태로 변환이 필요합니다.
400 (요청변수확인)	내용을 입력하세요.	답변내용(content)을 입력했는지 확인합니다. 
400 (요청변수확인)	동일한 답변이 존재합니다.	동일한 inquiryId 에 중복으로 답변을 입력했는지 확인합니다. 중복 답변 입력 시 에러가 발생합니다.
